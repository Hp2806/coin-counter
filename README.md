## Coin Detection and Counting using OpenCV

A computer vision project that detects and counts coins in an image using Python and OpenCV. The program lets the user select a coin image, applies thresholding and morphological operations to isolate the coins, then uses watershed segmentation to separate touching or overlapping coins and counts them accurately.
Features


## Features

- Interactive file picker for selecting coin images
- Automatic thresholding using Otsu's method
- Noise reduction using morphology operations
- Contour detection and labeling of each coin
- Displays total coin count on the output image



## Technologies Used

- Python
- OpenCV
- NumPy
- Tkinter
